Drawing a Line

pen red # Sets the pen color
fd 100 # Move the turtle forward

Tasks

  1. Try changing the line color or length.
  2. Make more than one line. Use different colors!

Extra

Did you know? You can do

speed 100 # Move super fast
pen red
fd 100

Making a Square

pen red
for [1..4]
  fd 100
  rt 90
