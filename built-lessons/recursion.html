Using a Function Inside Itself

This is called recursion.

square = (n) ->
  pen red
  for [1..4]
    fd n
    rt 90
  if n > 0 # Why do we need to check this?
    square n - 10 # Use the square function inside itself

square 100 # Why does this make 10 squares?
