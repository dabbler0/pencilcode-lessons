CSS

Stands for Cascading Style Sheets. Remember the write function?

words = write 'Hello Everyone!'

That's boring though, let's do something flashier:

words = write 'Hello Everyone!'
words.css {
  'color': 'brown'
  'background-color': 'yellow'
  'font-family': 'Courier New' # font
  'font-size': '30px'
  'border': '3px solid blue'
  'border-radius': '5px' # rounded corners
}

You can find a list of css properties here: http://www.w3schools.com/cssref/.

Extra

  1. Did you know? You can also retrieve css properties:

write words.css('font-family') # get css property
